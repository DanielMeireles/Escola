Cadastrar Usuário

Ao acessar a janela "Cadastrar Usuário" é possível cadastrar um usuário na base de dados.

Acesso à janela "Cadastrar Usuário"

  1. Clique no item "Usuário" no menu superior da janela principal;

  2. Acesse o item "Cadastrar Usuário".

Cadastrar Usuário

  1. Informe corretamente o usuário no campo especificado;

  2. Informe corretamente a senha no campo especificado;

  3. No campo especificado, selecione o perfil de usuário ("Administrador" ou "Operacional");

  4. Clique no botão"Salvar" para finalizar o cadastro.

Observações

  1. Para limpar todos os campos da tela, clique no botão "Limpar";

  2. Para sair da tela, clique no botão "Sair".
