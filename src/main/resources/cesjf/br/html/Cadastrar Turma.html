Cadastrar Turma

Ao acessar a janela "Cadastrar Turma" é possível cadastrar uma turma na base de dados.

Acesso à janela "Cadastrar Turma"

  1. Clique no item "Cadastro" no menu superior da janela principal;

  2. Acesse o item "Cadastrar Turma".

Cadastrar Turma

  1. Informe corretamente o nome da turma no campo especificado;

  2. No campo especificado, informe corretamente o ano em que a turma será criada;

  3. No campo especificado, selecione o tipo de ensino ("Fundamental I", "Fundamental" ou "Médio");

  4. Clique no botão"Cadastrar" para finalizar o cadastro.

Observações

  1. Para limpar todos os campos da tela, clique no botão "Limpar";

  2. Para sair da tela, clique no botão "Sair".
