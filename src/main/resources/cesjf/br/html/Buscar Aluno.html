Buscar Aluno

Ao acessar a janela "Buscar Aluno" é possível buscar um aluno cadastrado na base de dados.

Acesso à janela "Buscar Aluno"

  1. Clique no item "Busca" no menu superior da janela principal;

  2. Acesse o item "Buscar Aluno".

Buscar Aluno

  1. Informe corretamente a matrícula do aluno no campo especificado;

  2. Clique no botão"Pesquisar" para efetuar a pesquisa.

  3. Os campos ("Nome", "Ano Nascimento", "PCD" e "Turma") especificados serão preenchidos com os dados do aluno pesquisado;

Observações

  1. Para habilitar todos os campos para edição, clique no botão "Editar";

  2. Para salvar todas as edições feitas, clique no botão "Salvar";

  3. Para remover o aluno da base de dados, clique no botão "Remover";

  4. Para sair da tela, clique no botão "Sair".
