Buscar Alunos

Ao acessar a janela "Buscar Alunos" é possível buscar todos os alunos cadastrados na base de dados.

Acesso à janela "Buscar Alunos"

  1. Clique no item "Busca" no menu superior da janela principal;

  2. Acesse o item "Buscar Alunos".

Buscar Alunos

  1. Informe corretamente a turma no campo especificado;

  2. Clique no botão"Buscar" para efetuar a busca.

  3. Na tabela especificada, serão listados todos os alunos referentes a determinada turma;

  4. Selecione um aluno na tabela, em seguida clique no botão "Aluno" para exibir a janela "Busca Aluno;" com todas as informações referentes a determinado aluno.

Observação

  1. Para sair da tela, clique no botão "Sair".
