Buscar Usuário

Ao acessar a janela "Buscar Usuário" é possível visualizar todos os usuários cadastrados ou efetuar a pesquisa de um usuário específico.

Acesso à janela "Buscar Usuário"

  1. Clique no item "Usuário" no menu superior da janela principal;

  2. Acesse o item "Buscar Usuário".

Buscar Usuário

  1. Na tabela serão listados todos os usuários cadastrados;

  2. Para pesquisar um determinado usuário, informe corretamente o nome no campo especificado;

  3. Clique no botão"Pesquisar" para efetuar a pesquisa;

  4. Caso encontrado, o usuário será listado na tabela.

Observações

  1. Para alterar os dados de um determinado usuário, selecione-o na tabela e clique no botão "Alterar Usuário";

  2. Para excluir um determinado usuário, selecione-o na tabela e clique no botão "Excluir";

  3. Para sair da tela, clique no botão "Sair".
