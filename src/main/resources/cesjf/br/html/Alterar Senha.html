Alterar Senha

Ao acessar a janela de "Alterar Senha" é possível alterar a senha de um usuário cadastrado na base de dados.

Acesso à janela "Alterar Senha"

  1. Clique no item "Usuário" no menu superior da janela principal;

  2. Acesse o item "Alterar Senha".

Alterar Senha

  1. Informe corretamente a senha atual do usuário;

  2. Informe a nova senha do usuário;

  3. Informe novamente a nova senha do usuário para que seja confirmada;

  4. Clique no botão"Alterar" para finalizar a alteração.

Observação

  1. Para sair da tela, clique no botão "Sair".
