Buscar Turmas

Ao acessar a janela "Buscar Turmas" é possível buscar as turmas cadastradas na base de dados, além da opção de filtrar a pesquisa por PCD e visualizar as informações dos alunos das turmas.

Acesso à janela "Buscar Turmas"

  1. Clique no item "Busca" no menu superior da janela principal;

  2. Acesse o item "Buscar Turmas".

Buscar Turmas

  1. Na tabela especificada, será listada todas as turmas cadastradas;

  2. Selecione a opção "Sim" para buscar todas as turmas com ao menos um aluno PCD;

  3. Selecione a opção "Não" para buscar todas as turmas;

  4. Clique no botão"Alunos Turma" para exibir a janela "Buscar Alunos;" com todos os alunos de determinada turma.

Observação

  1. Para sair da tela, clique no botão "Sair".
