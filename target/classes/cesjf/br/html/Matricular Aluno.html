Matricular Aluno

Ao acessar a janela "Matricular Aluno" é possível cadastrar um aluno na base de dados.

Acesso à janela "Matricular Aluno"

  1. Clique no item "Cadastro" no menu superior da janela principal;

  2. Acesse o item "Matricular Aluno".

Matricular Aluno

  1. No campo especificado, selecione a turma em que o aluno será matriculado;

  2. Informe corretamente a matrícula do aluno no campo especificado;

  3. Informe corretamente o nome do aluno no campo especificado;

  4. Informe corretamente o ano de nascimento do aluno no campo especificado;

  5. No campo especificado, selecione a opção PCD ("Não" ou "Sim");

  6. Clique no botão"Cadastrar" para finalizar a matrícula.

Observações

  1. Para limpar todos os campos da tela, clique no botão "Limpar";

  2. Para sair da tela, clique no botão "Sair".
